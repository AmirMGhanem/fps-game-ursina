# Multiplayer FPS Game

- Multiplayer FPS game using Ursina Engine (Python)
- Server made with javascript (easier to work with websocket)

## How to use:

- Setup `.env` (with `.env.template` example)

### Server:
```bash
  npm i && npm start
```
### Client:
```bash
  pip install -r requirements.txt
  python main.py
```
